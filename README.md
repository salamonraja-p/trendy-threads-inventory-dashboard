🏆 Trendy Threads – Inventory Management System
🥇 Buildathon First Prize Winning Project

📌 Project Overview
Trendy Threads is a full‑stack inventory management and sales dashboard system developed for a clothing store.
The application helps manage products, stock, sales, and trending items through a clean and user‑friendly interface.

This project won 🏆 First Prize in a Buildathon, showcasing real‑world full‑stack development skills.

🚀 Features

Product listing with images
Stock and sales tracking
Trending products display
Search and sorting functionality
Dashboard with business insights
Responsive UI

Recommended size: 800 × 600
Images served from Spring Boot static folder
